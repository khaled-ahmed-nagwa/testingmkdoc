    The getSavedUsers Function Path

    the getSavedUsers is located at the lib/features/authentication/data/local/data_sources/authentication_local_data_source_impl.dart path.

    getSavedUsers Function Return

    Function Return List of UserEntity objects that contains:
    • userID
    • portalId
    • username
    • firstName
    • lastName
    • status
    • roles
    • phoneNumber
    • countryPhoneCode
    • language
    • country
    • portalType
    • imageUrl
    • portalCurrency
    • portalTimeZone
